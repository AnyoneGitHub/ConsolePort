## Recommended software for Xbox 360 on Windows:
## JoyToKey (best mouse emulation, simplistic setup, shareware)
## http://joytokey.net/en/
## Instructions:
##  1. Unzip JoyToKey anywhere
##  2. Do NOT start JoyToKey!
##  3. Place the profile in the program directory
##  4. Start JoyToKey
